# Speed to Lead v5 — Progress Notes

This folder tracks what's been done, what's working, what's broken, and where to resume.

## How to use this

- Each file = one phase or session.
- `_STATUS.md` = the current state of the project. Start here.
- Phase files (e.g. `01_COMPLETED_PHASE_0.md`) = what was done, what tests were added, any gotchas.
- `NEXT_SESSION.md` = where to start next time.

## Legend

- ✅ = done and verified
- ⚠️ = in progress / partially done
- ❌ = known broken, not fixed yet
- 🔲 = not started
